# Iris Flower Classification

## Overview
This project classifies Iris flowers into three species - Setosa, Versicolor and Virginica based on their sepal and petal measurements.

## Dataset
The classic Iris dataset containing 150 samples with sepal length, sepal width, petal length, petal width and species columns.

## Approach
1. Data Exploration and Visualization
2. Exploratory Data Analysis (EDA) with pairplots, boxplots and histograms
3. Correlation Analysis
4. Model Building using multiple algorithms - Logistic Regression, Decision Tree, KNN, SVM and Random Forest
5. Model Evaluation and Comparison
6. Confusion Matrix and Feature Importance Analysis
7. Prediction on new flower measurements

## Results
All models achieved high accuracy on this dataset, with most reaching 100% on the test set due to the clear separability of the Iris species.

## Files
- `Iris_Flower_Classification.ipynb` - Complete code with analysis
- `IRIS.csv` - Dataset used
