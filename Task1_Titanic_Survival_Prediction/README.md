# Titanic Survival Prediction

## Overview
This project predicts whether a passenger survived the Titanic disaster based on features like age, gender, ticket class, fare and family size.

## Dataset
The dataset contains information about 891 passengers including their age, gender, ticket class, fare, number of siblings/parents aboard and survival status.

## Approach
1. Data Cleaning and Handling Missing Values
2. Exploratory Data Analysis (EDA)
3. Feature Engineering (Family Size, IsAlone)
4. Label Encoding categorical features
5. Feature Scaling
6. Model Building using multiple algorithms - Logistic Regression, Decision Tree, KNN, SVM, Random Forest and Gradient Boosting
7. Model Evaluation and Comparison
8. Confusion Matrix and Feature Importance Analysis

## Results
Random Forest and Gradient Boosting performed the best among all models tested, achieving the highest accuracy on the test dataset.
