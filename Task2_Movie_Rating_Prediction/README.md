# Movie Rating Prediction with Python

## Overview
This project predicts the rating of Indian movies based on features like genre, director, actors, votes and duration using regression techniques.

## Dataset
The dataset contains information about Indian movies including their name, year, duration, genre, rating, votes, director and main actors.

## Approach
1. Data Cleaning and Handling Missing Values
2. Exploratory Data Analysis (EDA) with multiple visualizations
3. Feature Engineering using Label Encoding
4. Model Building using multiple algorithms - Linear Regression, Random Forest Regressor and Gradient Boosting Regressor
5. Model Evaluation using MSE and R² Score
6. Feature Importance Analysis
7. Prediction on new movie data

## Results
Random Forest and Gradient Boosting outperformed Linear Regression, capturing non-linear relationships between features and movie ratings more effectively.

## Files
- `Movie_Rating_Prediction.ipynb` - Complete code with analysis
- `IMDb Movies India.csv` - Dataset used
